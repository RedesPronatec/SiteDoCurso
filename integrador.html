Projeto Integrador I

O projeto integrador visa orientar o educando para a elaboração de um trabalho onde o aluno deverá utilizar as ferramentas adquiridas nos demais componentes curriculares, exercitando a interdisciplinaridade e, procurando sempre a orientação e apoio técnico dos educadores.

Mr. Pharma

A Mr. Pharma foi um projeto desenvolvido por Camylle Lima, Tallane Silva, Suemy Silva e João Almeida. Com o intuito de produzir um sistema básico para farmácias usando a linguagem Portugol.

Mevely Express

Mevely foi um projeto desenvolvido por Jhuan Barbosa e Maria de Brito. Construiram um código para cadastro de alimentos em um supermercado.

Jogo da Velha

O jogo da velha foi um projeto desenvolvido por Miguel Gomes, Fabrícia Vieira, Marlon Lima, Guylherme Silva e Diego Silva. Um Jogo divertido usando Portugol.

Campo Minado

Campo Minado foi um projeto desenvolvido por Pedro Bomfim, Brygida Santos, Erick Silva, Jefferson Marinho e Welligton Cordeiro. Desenvolveram o código do jogo Campo Minado, interaativo e divertido.
